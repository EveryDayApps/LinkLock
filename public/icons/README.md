# Extension Icons

Place your extension icons in this directory:

- `icon-16.png` - 16x16 px (toolbar icon)
- `icon-32.png` - 32x32 px (toolbar icon @2x)
- `icon-48.png` - 48x48 px (extension management)
- `icon-128.png` - 128x128 px (Chrome Web Store)

All icons should be PNG format with transparent backgrounds.

You can use tools like:
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/
- Or design your own using Figma/Sketch/Photoshop

For now, the extension will work without icons, but browsers will show a default placeholder.
